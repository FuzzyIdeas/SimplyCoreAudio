The following classes are available globally.

  • AudioDevice

    This class represents an audio device managed by Core Audio.

    Devices may be physical or virtual. For a comprehensive list of supported types, please refer to TransportType.

  • AudioObject

    This class represents a Core Audio managed audio object. In Core Audio, audio objects are referenced by its AudioObjectID and belong to a specific AudioClassID.

    For more information, please refer to Core Audio’s documentation or source code.

  • SimplyCoreAudio

    This class provides convenient audio hardware-related functions (e.g. obtaining all devices managed by Core Audio) and allows audio hardware-related notifications to work. Additionally, you may create and remove aggregate devices using this class.

    Important

    If you are interested in receiving hardware-related notifications, remember to keep a strong reference to an object of this class.
