public class AMAudioObject: NSObject

AMAudioObject

This class represents a Core Audio object currently present in the system. In Core Audio, audio objects are referenced by its AudioObjectID and belong to a specific AudioClassID. For more information, please refer to Core Audio’s documentation or source code.
