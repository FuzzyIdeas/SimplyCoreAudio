Protocols

The following protocols are available globally.

AMNotificationCenter Protocols
  • AMEvent

    The protocol that any events must implement.

    An event conforming to this protocol may be a class, a struct or an enum. In AMCoreAudio, we will be relying on enums, since they are very lightweight yet expressive enough (we can pass arguments to them.)

    Declaration

    Swift

    public protocol AMEvent {}
  • AMEventSubscriber

    The protocol any event subscriber must implement.

    Typically, this will be a class that also happens to conform to the Hashable protocol.
